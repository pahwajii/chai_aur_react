1. Syntax Recap
jsx
Copy code
const [stateVariable, setStateFunction] = useState(initialValue);
useState() is a React (or Preact) hook used to manage local component state.

Returns an array:

stateVariable: the current state value

setStateFunction: the function used to update it

const [count, setCount] = useState(0);
count is the state variable initialized to 0.

setCount is the function that can update the count.

This could be used in future to track clicks, members, or any dynamic UI behavior.

Toggle UI elements

Store form inputs

Manage counters or scores

Trigger re-render on state change

Control component conditions (like showing/hiding components)

ğŸ”¹ 5. Rules of useState
Can only be used inside React/Preact function components

Must be called at the top level (not inside loops or conditions)

You can have multiple useState hooks in one component
